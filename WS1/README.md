## Intro

My brother is a daily user of [Drivy](https://www.drivy.com/), a french peer-to-peer car rental service.

He lists their 3 cars:

* Peugeot 306
* Range Rover Sport
* Porsche Boxter

Any person, call the `driver` can book a car for given dates/distance.
